# NIRBHAYA - Womens Safety Platform
nirbhaya is a Womens Safety Web Application developed by using HTML,CSS and javascript.

## FEATURES 
Emergency SOS Button
Live Location Tracking 
Trusted Contracts Management 
Emergency Helpline Numbers
Women's Safety Tips
Responsive User Interface

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Purpose

The purpose of this project is to provide women with quick access to emergency assistance, safety information, and trusted contacts during critical situations.
